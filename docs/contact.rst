Logging Issues with Carbon
~~~~~~~~~~~~~~~~~~~~~~~~~~

First, verify that the issue you are facing has not been logged, by
looking at unresolved `carbon issue
<https://projects.engineering.redhat.com/issues/?filter=-5&jql=project%20%3D%20CIDER%20AND%20component%20%3D%20Carbon%20AND%20resolution%20%3D%20Unresolved%20order%20by%20priority%20DESC%2Cupdated%20DESC>`_.

If the issue has not been logged, please log the issue using the following
options in `JIRA <https://projects.engineering.redhat.com/secure/CreateIssue!default.jspa>`_:

  * Project = CIDER(CID)
  * Component/s = Carbon
